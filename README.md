#  Lumicore

Site institucional fictício desenvolvido para apresentar os serviços de uma empresa de tecnologia, utilizando apenas **HTML5** e **CSS3**. O projeto foi criado com o objetivo de praticar desenvolvimento Front-end, organização de código, design responsivo e criação de interfaces modernas.

---

##  Funcionalidades

* Página Inicial (Home)
* Página Sobre
* Página Soluções
* Página Serviços
* Página Contato
* Formulário de contato (interface)
* Layout moderno
* Paleta de cores personalizada
* Design responsivo
* Navegação entre páginas

---

##  Tecnologias Utilizadas

* HTML5
* CSS3
* Flexbox
* Variáveis CSS
* Google Fonts
* Git
* GitHub
